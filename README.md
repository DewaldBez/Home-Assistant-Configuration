# Home Assistant Configuration

This is my personal Home Assistant setup. It is hosted on a Ubuntu virtual machine that is running [Home Assistant](https://hub.docker.com/r/homeassistant/home-assistant/) and [Let's Encrypt](https://hub.docker.com/r/linuxserver/letsencrypt) docker containers. 

Here are some of the benefits that HA provides me:
- Completely automated lighting
- DIY smart alarm system.
- DIY Nest-like thermostat
